# Placement

Whole slides anchor via the slide directive — `<!-- slide: .center .middle -->`,
`.bottom .right`, and so on. Individual boxes use `::: {.place …} :::`, any number
per slide. The nine labels below are each a placed box:

::: {.place .top .left}
`.top .left`
:::

::: {.place .top .center}
`.top .center`
:::

::: {.place .top .right}
`.top .right`
:::

::: {.place .middle .left}
`.middle .left`
:::

::: {.place .center .middle}
`.center .middle`
:::

::: {.place .middle .right}
`.middle .right`
:::

::: {.place .bottom .left}
`.bottom .left`
:::

::: {.place .bottom .center}
`.bottom .center`
:::

::: {.place .bottom .right}
`.bottom .right`
:::
