# Defense in Depth

Reveal these one click at a time:

::: incremental

- **Secure the weakest link** — attackers don't fight fair
- **Fail securely** — a crash should not become a bypass
- **Least privilege** — give each agent only the reach it needs
- **Be reluctant to trust** — every untrusted path can carry instructions

:::
